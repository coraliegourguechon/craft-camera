README 

Craft Camera Version 1 is a prototype.
You can find in this folder the template and note to build your own case.
The template is released as .EPS, to allow people to correct the case if needed.
Please take note that Craft Camera is a work on progress, so you might have to make
some adjustments. It is also based on Do It Yourself process, and do not aim to reach 
the quality of industrial products.
